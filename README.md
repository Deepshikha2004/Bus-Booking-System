# Bus Booking System

A comprehensive Bus Booking System built using **Python (Tkinter)** for the GUI and **MySQL** for database management. This system allows users to book bus tickets, view available buses, manage bookings, and perform administrative operations.

## Features

- User-friendly GUI built with Tkinter
- Book bus tickets
- View available buses and schedules
- Admin panel for managing buses and bookings
- Database integration using MySQL
- Input validation and error handling
- Search functionality for buses by route and date

## Technologies Used

- **Python 3.x** (Core Logic)
- **Tkinter** (Graphical User Interface)
- **MySQL** (Database Management)

## Prerequisites

Ensure you have the following installed:

- Python 3.x
- MySQL Server

You can install required Python packages using:

```bash
pip install mysql-connector-python
```

## Database Setup

1. Create a database in MySQL.
2. Use the created database.
3. Create required tables.

## How to Run the System

1. Clone the repository:

```bash
git clone https://github.com/Deepshikha2004/bus-booking-system.git
cd bus-booking-system
```

2. Ensure MySQL server is running and tables are created.

3. Run the main application:

```bash
python test1.py
```

## System Workflow

1. **User Interface:**
   - Search for buses by source and destination
   - Book tickets by selecting the bus and providing passenger details

2. **Admin Panel:**
   - Add new buses
   - View and manage bookings
